3D fluids simulations and Snow Simulation

Right now, we have successfully simulated the replusion and viscosity force through density and pressure computation.

It is commonsense to know that we can not let the particle to get too close because then all the particles may migrate together and overlap and the particles around each other should exert force on each other. Therefore, we need to have circle to store all the neighbors for a single particle at each iteration/timestamp. For all those neighbors, we have to calculate all the density and pressure. With those two measurements, we can easily to apply force on each each, in addition to the external force like gravity and collision, wind.

To faciliate us with faster computation, we came up with a particle structure to store all the needed properties.

struct Particle {

glm::dvec3 p;

glm::dvec3 v;

glm::dvec3 force;

std::vector neighbors;

double pressure;

double density; };

Normally, we should have a accelaration for verlet integration for new posistion. However, all the fluid particles are same heavy. Therefore, the force and accelaration are interchangeable.
